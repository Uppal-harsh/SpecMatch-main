# 🛒 SpecMatch

**Find Your Perfect Tech Companion — Fast, Smart, and Data-Driven.**

SpecMatch is a high-end comparison and recommendation engine for modern tech. Whether you are looking for a flagship smartphone, a professional laptop, or building a custom PC, SpecMatch helps you cut through the marketing noise and find the device that fits your exact needs.

![SpecMatch Hero Banner](https://images.unsplash.com/photo-1550745679-5652175bdd48?auto=format&fit=crop&q=80&w=2070)

## ✨ Key Features

*   **🧠 Intelligent Matching**: Answer a few simple questions about your usage patterns, and our algorithm recommends the best-fitting devices.
*   **📊 Cross-Store Comparison**: Real-time pricing and availability across 5 major retailers. 🤑
*   **🛠️ PC Builder**: A streamlined builder for enthusiasts to pick components without compatibility headaches.
*   **📂 CSV-Powered Data**: High-speed device data processing and ingestion using PapaParse.
*   **💨 Minimalist Performance**: Built with Next.js and Framer Motion for a blazing-fast, smooth-as-silk user experience.
*   **📱 Fully Responsive**: Optimized for desktop, tablet, and mobile viewing.

## 🚀 Technologies Used

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Data Parsing**: [PapaParse](https://www.papaparse.com/)

## 🛠️ Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Uppal-harsh/SpecMatch.git
    cd SpecMatch
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open your browser**:
    Navigate to [http://localhost:3000](http://localhost:3000) to see the app in action.

## 📁 Project Structure

- `/app`: Next.js App Router pages and layouts.
- `/components`: Reusable UI components (Hero, FlowingMenu, etc.).
- `/lib`: Global state (Zustand), utility functions, and API helpers.
- `/public`: Static assets (banners, icons, etc.).
